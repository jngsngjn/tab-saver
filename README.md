# Tab Saver

현재 열려 있는 모든 탭을 한 번에 저장하고,  
필요할 때 다시 열 수 있는 **심플한 Chrome 확장 프로그램**입니다.

---

## ✨ 기능

- 현재 창의 모든 탭 저장
- 저장한 탭 한 번에 복원
- 새 창에서 복원 옵션 지원
- 로컬 저장 (계정 / 클라우드 없음)
- `chrome://`, `about:blank` 탭 자동 제외

---

## 🧭 사용법

### 탭 저장
1. Tab Saver 아이콘 클릭
2. **[현재 탭 저장]** 클릭
3. → `탭 N개 저장됨` 메시지 확인

### 탭 복원
- **현재 창에서 복원**
    - 체크박스 해제 후 **[탭 다시 열기]**

- **새 창에서 복원**
    - ☑️ *새 창에서 복원* 체크 후 **[탭 다시 열기]**

---

## 🛠 설치 (개발자 모드)

1. `chrome://extensions` 접속
2. 개발자 모드 ON
3. **압축 해제된 확장 프로그램 로드**
4. 프로젝트 폴더 선택
